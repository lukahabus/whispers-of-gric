# NERO - Whispers of Grič

Povijesni murder-mystery / 2D point-and-click avantura smještena u Zagreb 18. stoljeća, u podijeljeni grad između **Gradeca** i **Kaptola**.

Igraš **Neru**, istražiteljicu bez korijena i bez frakcijske strane. Žene optužene za vještičarenje nestaju, a netko koristi strah od vještica kao paravan za niz ubojstava - dok sukob Gradeca i Kaptola služi kao dimna zavjesa. Simbol svijeće vodi te iz tame prema istini: **NERO znači tama, a igra završava svjetlom.**

## Kanonska verzija

**`nero.html`** je kanonska, igriva verzija igre (NERO: Whispers of Grič). Otvori je u pregledniku.

- Tri čina + epilog: **Nestanak → Istraga → Obračun → NERA**
- Istraga, dijalozi, frakcije (Gradec/Kaptol), inventar s kombiniranjem
- Scene skrivenih predmeta, zagonetka „Svijeće nevinih”, preklapanje karata
- Četiri završetka, te svijet koji postaje svjetliji kako otkrivaš istinu

## Datoteke

- `nero.html` - **kanonska igra** (NERO)
- `prezentacija.html` - pitch deck (NERO); ← → za navigaciju, `T` timer, `F` fullscreen
- `new assets/` - aktualni vizuali, portreti i karte
- `assets/` - pomoćni/stariji vizuali koje igra još koristi
- `index.html` - ranija varijanta („Grička Sjena”, stranac + ukradena ogrlica); zadržana kao arhiva, nije kanonska
- `gricka_sjena_gdd.pdf` - game design dokument
- `nero.txt` - bilješke priče i zagonetki

## Likovi

- **Nera** - istražiteljica (igrač)
- **Mudra starica s Kaptola** - vodič koji govori istinu samo u zagonetkama
- **Redovnik Blaž** - čuvar kaptolskog arhiva
- **Zlatar Juraj** - knjiga s dva unosa za istu narudžbu
- **Knez Drašković** - plemić iznad obje strane; sjena iza svega
